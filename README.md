# 每日总结

一款轻量级的每日总结桌面应用，使用 Python + Tkinter 构建，数据本地存储，无需联网。

## 功能特性

- 📅 日历导航 — 点击选择日期，月份自由切换
- ✏️ 文本编辑 — 多行输入，自动加载历史内容
- 💾 本地存储 — 数据按月保存为 JSON 文件，隐私安全
- 🔖 智能标记 — 有总结的日期显示绿色小圆点
- ⌨️ 快捷保存 — `Ctrl+S` 一键保存

## 快速开始

### 环境要求

- Python 3.8+
- 无需安装第三方依赖（仅使用标准库）

### 运行

```bash
git clone https://github.com/stevendong666/daily-summary.git
cd daily-summary
python main.py
```

## 数据存储

总结数据保存在 `data/` 目录下，按月存储：

```
data/
├── 2026_05.json
├── 2026_06.json
└── ...
```

文件格式：

```json
{
  "2026-05-08": "今天完成了项目重构...",
  "2026-05-09": "修复了日历显示的 bug..."
}
```

## 技术栈

| 组件 | 技术 |
|------|------|
| GUI | Tkinter (Python 标准库) |
| 日历 | calendar (Python 标准库) |
| 存储 | JSON 文件 |
